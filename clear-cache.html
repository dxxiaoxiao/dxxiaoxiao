清除浏览器缓存指南

如果您看到的不是最新版本的待办事项应用，可能是由于浏览器缓存了旧版本的文件。请按照以下步骤清除浏览器缓存：

Chrome

  1. 按下 Ctrl+Shift+Delete (Windows) 或 Command+Shift+Delete (Mac)
  2. 时间范围选择"过去1小时"
  3. 勾选"缓存的图片和文件"
  4. 点击"清除数据"

Firefox

  1. 按下 Ctrl+Shift+Delete (Windows) 或 Command+Shift+Delete (Mac)
  2. 时间范围选择"最近1小时"
  3. 勾选"缓存"
  4. 点击"立即清除"

Edge

  1. 按下 Ctrl+Shift+Delete
  2. 时间范围选择"过去1小时"
  3. 勾选"缓存的图片和文件"
  4. 点击"立即清除"

Safari

  1. 点击菜单中的"Safari"
  2. 选择"清除历史记录和网站数据..."
  3. 选择"最近一小时"
  4. 点击"立即清除"

提示： 清除缓存后，请刷新页面或点击下方按钮访问待办事项应用。如果仍然看到旧版本，请尝试使用隐私浏览模式打开应用。
